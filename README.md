# UK Wesley Website

This is a GitHub Pages-ready rebuild of the UK Wesley homepage.

## Image paths needed

Place these image files in:

`media/uploads/`

- `uk-wesley-logo.png`
- `ukwf-hero.png`
- `ukwf-worship.png`
- `ukwf-dorm.png`

You already have this one:
`media/uploads/ukwf-worship.png`

The page is mobile-friendly and includes dropdown submenus for desktop and tap-to-open submenus for mobile.
